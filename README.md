# CodeClaw - AI编程助手平台

## 项目定位
面向海外开发者的AI编程助手，集成国产大模型（DeepSeek/GLM/Kimi），提供Claude 1/10价格的替代方案。

## 核心功能
1. 多模型切换（DeepSeek-V3、GLM-4、Kimi-K2）
2. 代码生成与解释
3. 按Token计费，预充值模式
4. Web IDE界面

## 技术栈
- 前端：Vue 3 + TypeScript
- 后端：Node.js + Express
- 数据库：PostgreSQL
- 部署：Vercel + Railway
